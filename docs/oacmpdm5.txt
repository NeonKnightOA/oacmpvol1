=====================
oajgdm1 - The Other Side
=====================

Title:			The other side
Date:			04-02-2013
Author:			Jan Groothuijse (http://jan.groothuijse.net/maps.html) (jan@groothuijse.net)
File:			oajgdm1.pk3
Type:			Deathmatch: FFA, Tourney, TDM
Players:		2 - 6

Description:	
	Open 'outdoor' map. Designed so that there would be room to fight. Redundant teleporters where added to make chases more interessting.

Tools:
	NetRadiant 1.5.0n-git806e3fe
	using (among others) Q3Map2 options: -meta -vis -bound 8
	
	bspc	(to generate aas file)
	using "~/bspc -bsp2aas oajgdm1.bsp -optimize -forcesidesvisible"
